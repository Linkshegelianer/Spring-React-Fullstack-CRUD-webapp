##  Technologies
#### Backend:
1. Java 20 + Gradle 8.2
2. Spring Boot + Spring Data JPA + Spring Security
3. JSON Webtoken
4. Liquibase
5. Rollbar
5. JUnit/AssertJ
6. Querydsl
7. Swagger

#### Frontend:
1. React 17.0.2
2. Bootstrap
3. Sass
3. Babel
4. Axios

#### Databases:
1. H2
1. PostgreSQL
